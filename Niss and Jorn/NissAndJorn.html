ICT Ingeniør

Hej og velkommen til min sektion af hjemmesiden!

Min nysgerrighed inden for IT har altid eksisteret, jeg har leget med computere så længe jeg kan huske, derfor er jeg ekstatisk for at være med til at starte NiVes op sammen med Asger.

Som IT ingeniør har jeg specialisation i Cross-Media, det betyder at jeg har haft fingrene i alt lige fra Android apps og Web applications til store systemer med advancerede databaser og logik!

Jeg har lige været i praktik i Holland hvor jeg var med til at skabe en potentiel gigantisk web application inden for endoscoper i det medicinale område. Da jeg desværre blev nød til at lægge det bag mig pga studier, besluttede jeg mig for at starte NiVes så jeg kunne få lov til at lave det jeg har passion for. 
Jeg håber jeg kan være med til at skabe lige præcis det produkt som du har brug for og sørge for at dit firma bliver repræsenteret bedst muligt på nettet!


Herunder kan du se et udtag af mine færdigheder inden for IT.

Web and Design

HTML og CSS kombineret i alle mulige karismatiske måder fra simple til meget advancerede. Det er her jeg rigtig for lov at udspille mig og kan sætte mit eller dit eget præg på produktet.

JavaScript er også inkluderet.



C#

I modsætning til Web design er C# et område hvor jeg går ind for optmering og ren kode. Det er vigtigt at logikken indenfor det produkt jeg laver kører så effektivt som muligt.

I dette område er jeg effektiv inden for Visual Studio fra Microsoft som giver uendelige muligheder inden for systemer.


Java

Java bliver brugt til at kode Android apps og Java applicationer, Java og C# ligner meget hinanden og begge er et felt jeg er meget sikker i.

Database

Databaser har jeg arbejdet meget med og har lavet dem i så mange forskellige omfang at jeg tager alle udfodringer med et smil.
